## 📋 SOBRE O PROJETO

Este é meu portfólio pessoal desenvolvido para apresentar minha jornada como estudante de **Engenharia de Software na FIAP**. O projeto foi construído com **HTML5, CSS3 e JavaScript puro**, sem dependências externas pesadas, priorizando performance, responsividade e uma experiência visual moderna.

### ✨ CARACTERÍSTICAS PRINCIPAIS

- **🌓 Dark/Light Mode** - Alternância entre temas com um clique
- **📱 Design Responsivo** - Adapta-se perfeitamente a celulares, tablets e desktops
- **🎨 Interface Moderna** - Efeitos glassmorphism, gradientes e animações suaves
- **🚀 Performance Otimizada** - Código leve e sem frameworks pesados
- **♿ Acessibilidade** - Suporte a prefers-reduced-motion e contraste adequado

## ⚡ ANIMAÇÕES E INTERAÇÕES

### Animações CSS
- `fadeInUp` - Elementos sobem suavemente
- `fadeInLeft` - Entrada pela esquerda
- `fadeInRight` - Entrada pela direita
- `scaleIn` - Cresce suavemente
- `float` - Flutuação contínua
- `shimmer` - Brilho deslizante nas barras
- `borderGlow` - Borda com brilho pulsante
- `gradientShift` - Gradiente animado

### Interações JavaScript
- **Scroll Reveal** - Elementos aparecem ao rolar
- **Active Link** - Destaque no menu conforme seção
- **Menu Mobile** - Hambúrguer menu funcional
- **Theme Toggle** - Alternância suave entre temas
- **Form Feedback** - Animação de sucesso ao enviar
